HTML and CSS

Here are some links to learn the basics about HTML and CSS. We will learn how to use specific HTML tags and CSS rules during the partice session. Be very careful if you browse the web on your own: a lot of ressources are outdated or do not show good practices. If you are looking at answers on StackOverflow or similar, please check the date of the answer,and ompare the number of votes on the accepted answer and the second answer.
Keep in mind:

  • HTML is for content only,
  • CSS is for presentation,
  • use tables in order to display tabular data, not for layout!
